1. Introduction to C

C is a general-purpose, procedural programming language used to develop system software, embedded systems, operating systems, compilers, and applications. It is fast, powerful, and the foundation of many other languages.

C is known as a middle-level language.

It combines features of both high-level and low-level languages.

C is one of the best languages to learn programming fundamentals, logic building, and memory concepts.

2. History of C

C was developed by Dennis Ritchie at Bell Labs in the early 1970s. It was originally designed for system programming and later became one of the most widely used languages in the world.

Topic Details
Creator Dennis Ritchie
Year 1972
Purpose System programming, OS development

3. C Compiler and Setup

To run C programs, you need a C compiler such as GCC, Clang, or Turbo C. Modern students mostly use GCC in VS Code, Code::Blocks, or online compilers. 

#include <stdio.h>

int main() {
    printf("Hello World");
    return 0;
}

Compiler

Translates C code into machine code.

IDE

Code::Blocks, Dev C++, VS Code, Turbo C.

Header File

Contains declarations for library functions.

4. Structure of a C Program

A C program generally contains header files, main function, declarations, logic, and return statement.

#include <stdio.h>

int main() {
    printf("Welcome to C");
    return 0;
}
Part Meaning
#include Includes library header files
main() Starting point of the program
printf() Prints output on screen
return 0 Indicates successful execution

5. Basic Syntax

C syntax is simple and structured. Statements end with semicolon, blocks are written in curly braces, and comments are used for explanation. 

// Single line comment

/* Multi
   line
   comment */

#include <stdio.h>

int main() {
    printf("C Syntax");
    return 0;
}

Semicolon ; ends a statement.

{ } defines a block of code.

6. Variables

Variables are used to store data values in memory.

#include <stdio.h>

int main() {
    int age = 20;
    float marks = 89.5;
    char grade = 'A';

    printf("%d %f %c", age, marks, grade);
    return 0;
}
Type Example Use
int int age = 20; Stores whole numbers
float float marks = 89.5; Stores decimal numbers
char char grade = 'A'; Stores a single character

7. Data Types

C has several built-in data types.

int a = 10;
float b = 5.5;
double c = 12.345;
char d = 'X';
Data Type Meaning
int Integer numbers
float Single precision decimal numbers
double Double precision decimal numbers
char Single character
void No value / no return type
C also supports modifiers like short, long, signed, and unsigned.

8. Operators

Operators are symbols used to perform operations on data.

int a = 10, b = 5;

a + b;
a - b;
a * b;
a / b;
a % b;
Operator Meaning
+ Addition
- Subtraction
* Multiplication
/ Division
% Modulus 
a > b;
a < b;
a == b;
a != b;
a >= b;
a <= b;

Arithmetic, relational, logical, assignment, and bitwise operators are all used in C.

9. Input and Output

C uses printf() for output and scanf() for input. These are defined in the stdio.h header file. 

#include <stdio.h>

int main() {
    int num;
    printf("Enter a number: ");
    scanf("%d", &num);
    printf("You entered: %d", num);
    return 0;
}
Function Use
printf() Print output
scanf() Take input
In scanf(), always use & before variable name for normal variables like int and float.

10. Conditional Statements

Conditions help the program make decisions.

#include <stdio.h>

int main() {
    int age = 18;

    if(age >= 18) {
        printf("Adult");
    } else {
        printf("Minor");
    }

    return 0;
}
int marks = 75;

if(marks >= 90) {
    printf("A");
} else if(marks >= 75) {
    printf("B");
} else {
    printf("C");
}
int day = 2;

switch(day) {
    case 1:
        printf("Monday");
        break;
    case 2:
        printf("Tuesday");
        break;
    default:
        printf("Other day");
}

11. Loops

Loops are used to repeat code multiple times.

for(int i = 1; i <= 5; i++) {
    printf("%d\n", i);
}
int i = 1;
while(i <= 5) {
    printf("%d\n", i);
    i++;
}
int j = 1;
do {
    printf("%d\n", j);
    j++;
} while(j <= 5);
Loop Use
for When number of repetitions is known
while Repeat while condition is true
do...while Runs at least once

12. Functions

Functions are reusable blocks of code that perform a specific task.

#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

int main() {
    printf("%d", add(5, 3));
    return 0;
}
Part Meaning
Function declaration Tells compiler about the function
Function definition Actual code written inside function
Return type Type of value returned
Functions help in code reuse, readability, and modular programming.

13. Arrays

Arrays store multiple values of the same data type in continuous memory.

int marks[5] = {90, 85, 78, 92, 88};

printf("%d", marks[0]);
Concept Meaning
Index Position of element, starts from 0
Size Total number of elements
Traversal Accessing all elements using loop 
for(int i = 0; i < 5; i++) {
    printf("%d ", marks[i]);
}

14. Strings

In C, strings are arrays of characters ending with a null character \0.

#include <stdio.h>

int main() {
    char name[] = "Shubham";
    printf("%s", name);
    return 0;
}
Item Meaning
char array Used to store string
\0 Null terminator
%s Format specifier for string

String handling in C is done using functions from string.h.

15. Pointers

Pointers store the memory address of another variable. This is one of the most important topics in C. 

#include <stdio.h>

int main() {
    int x = 10;
    int *ptr = &x;

    printf("Value: %d\n", x);
    printf("Address: %p\n", &x);
    printf("Pointer value: %p\n", ptr);
    printf("Pointer points to: %d\n", *ptr);

    return 0;
}
Symbol Meaning
& Address of operator
* Pointer declaration / dereference
ptr Pointer variable
Pointers are powerful but must be used carefully to avoid errors.

16. Storage Classes

Storage classes define scope, lifetime, and visibility of variables.

Storage Class Meaning
auto Default local variable
register Stores variable in CPU register if possible
static Retains value between function calls
extern Refers to global variable defined elsewhere

17. Structures

Structures allow grouping of different data types under one name.

#include <stdio.h>

struct Student {
    int roll;
    char name[20];
    float marks;
};

int main() {
    struct Student s1 = {1, "Shubham", 91.5};
    printf("%d %s %.2f", s1.roll, s1.name, s1.marks);
    return 0;
}
Part Meaning
struct Keyword to create structure
member Variable inside structure
. Used to access members

18. Unions

A union is similar to a structure, but all members share the same memory location.

union Data {
    int i;
    float f;
    char c;
};
In union, only one member can hold a value at a time.

19. Enumeration

enum Day {

Feature Use

20. File Handling

File handling is used to store and read data from files.

#include <stdio.h>

int main() {
    FILE *fp;
    fp = fopen("data.txt", "w");
    fprintf(fp, "Hello File");
    fclose(fp);
    return 0;
}
Function Use
fopen() Open a file
fprintf() Write to file
fclose() Close a file

21. Recursion

Recursion is when a function calls itself.

#include <stdio.h>

int fact(int n) {
    if(n == 0)
        return 1;
    return n * fact(n - 1);
}

Recursion needs a base condition to stop the function call.

22. Dynamic Memory Allocation

Dynamic memory allows memory to be allocated during program execution.

#include <stdio.h>
#include <stdlib.h>

int main() {
    int *ptr;
    ptr = (int*) malloc(5 * sizeof(int));

    if(ptr == NULL) {
        printf("Memory not allocated");
    } else {
        printf("Memory allocated");
    }

    free(ptr);
    return 0;
}
Function Use
malloc() Allocates memory
calloc() Allocates and initializes memory
realloc() Resizes allocated memory
free() Releases memory

23. Preprocessor Directives

Preprocessor commands are processed before compilation.

#include <stdio.h>
#define PI 3.14

int main() {
    printf("%f", PI);
    return 0;
}
Directive Use
#include Includes header files
#define Defines constants or macros
#ifdef Conditional compilation

24. C Best Practices

  • Use meaningful variable names.
  • Write small and clear functions.
  • Always initialize variables.
  • Use comments where needed.
  • Free dynamically allocated memory.
  • Check return values of functions.
  • Use indentation properly.
  • Test pointers carefully.

25. Important C Interview Questions

Question Answer
What is C? C is a procedural programming language used for system and application development.
Who developed C? Dennis Ritchie.
What is the use of printf? It prints output on the screen.
Difference between array and pointer? Array stores multiple values, pointer stores memory address.
What is a structure? A user-defined data type that groups different data types.
What is recursion? When a function calls itself.
Learn variables, operators, conditions, loops, functions, arrays, strings, pointers, structures, files, and memory management to become strong in C.

26. Full C Example

This example shows a simple interactive C program.

#include <stdio.h>

int main() {
    int num;
    printf("Enter a number: ");
    scanf("%d", &num);

    if(num % 2 == 0) {
        printf("Even Number");
    } else {
        printf("Odd Number");
    }

    return 0;
}

Quick recap: C is fast, structured, and very important for programming basics and memory understanding.
