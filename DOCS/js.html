1. Introduction to JavaScript

JavaScript is a programming language used to make web pages interactive and dynamic. It can change HTML, update CSS, handle events, work with APIs, and store data.

HTML gives structure

CSS gives style

JavaScript gives behavior

JavaScript runs in the browser and also on the server using Node.js.

2. How JavaScript Works

JavaScript is executed by the browser's JavaScript engine. It can be written directly inside HTML or in an external file. 

<script>
  console.log("Hello World");
</script>
<script src="app.js"></script>

Inline JS

Written directly in HTML using script tag.

External JS

Written in a separate .js file and linked to HTML.

3. JavaScript Syntax

JavaScript statements usually end with semicolons, but they are optional.

console.log("Hello");
alert("Welcome");

let name = "Shubham";
let age = 20;
Part Meaning
Statement A complete instruction
Variable Stores data
Function Reusable block of code

4. Variables

Variables store values that can be used later in the program.

var x = 10;
let y = 20;
const z = 30;
Keyword Meaning
var Old way, function scoped
let Modern, block scoped, can change
const Modern, block scoped, cannot be reassigned
Use let and const instead of var in modern JavaScript.

5. Data Types

JavaScript has different types of values.

let str = "Hello";
let num = 42;
let bool = true;
let arr = [1, 2, 3];
let obj = { name: "Shubham", age: 20 };
let undef;
let nul = null;
Type Example Meaning
String "Hello" Text
Number 42 Numeric value
Boolean true / false Logical value
Array [1,2,3] List of values
Object {} Key-value data
Undefined let x; Variable declared but not assigned
Null null Intentional empty value

6. Operators

Operators are used to perform actions on values.

let a = 10;
let b = 5;

a + b;
a - b;
a * b;
a / b;
a % b;
a ** b;
Operator Meaning
+ Addition
- Subtraction
* Multiplication
/ Division
% Remainder
** Power 
10 > 5;
10 < 5;
10 == 10;
10 === 10;
10 != 5;
10 !== "10";

== checks value only.

=== checks value and type.

7. Conditional Statements

Conditions help your code make decisions.

let age = 18;

if(age >= 18){
  console.log("Adult");
}else{
  console.log("Minor");
}
let marks = 75;

if(marks >= 90){
  console.log("A");
}else if(marks >= 75){
  console.log("B");
}else{
  console.log("C");
}
let day = "Monday";

switch(day){
  case "Monday":
    console.log("Start of week");
    break;
  default:
    console.log("Other day");
}

8. Loops

Loops repeat code multiple times.

for(let i = 1; i <= 5; i++){
  console.log(i);
}
let i = 1;
while(i <= 5){
  console.log(i);
  i++;
}
let j = 1;
do{
  console.log(j);
  j++;
}while(j <= 5);
Loop Use
for Known number of repetitions
while Repeat while condition is true
do...while Runs at least once

9. Functions

Functions are reusable blocks of code.

function greet(name){
  return "Hello " + name;
}

console.log(greet("Shubham"));
const greet = (name) => {
  return `Hello ${name}`;
};
Type Meaning
Function declaration Standard function syntax
Arrow function Short modern syntax
Return Sends result back
Functions help you avoid repeating code.

10. Arrays

Arrays store multiple values in one variable.

let fruits = ["apple", "mango", "banana"];

console.log(fruits[0]);
fruits.push("orange");
fruits.pop();
fruits.shift();
fruits.unshift("kiwi");
Method Use
push() Add to end
pop() Remove from end
shift() Remove from start
unshift() Add to start
length Array size 
let nums = [1,2,3,4,5];
nums.map(n => n * 2);
nums.filter(n => n > 2);
nums.reduce((a,b) => a + b);

11. Objects

Objects store data in key-value pairs.

let person = {
  name: "Shubham",
  age: 20,
  city: "Kota"
};

console.log(person.name);
Part Meaning
Key Name of the property
Value Data stored in the key

Objects are used for real-world data like users, products, settings, and more.

12. Strings

Strings are text values.

let text = "Hello World";
let len = text.length;
let upper = text.toUpperCase();
let lower = text.toLowerCase();
let part = text.slice(0, 5);
let replaced = text.replace("World", "JS");
Method Use
length String size
toUpperCase() Convert to capital letters
toLowerCase() Convert to small letters
slice() Extract part of string
replace() Replace text

13. Numbers and Math

JavaScript has built-in Math functions.

let n = 10.789;

Math.round(n);
Math.floor(n);
Math.ceil(n);
Math.random();
Math.max(10, 20, 5);
Math.min(10, 20, 5);

14. Scope

Scope tells where a variable can be accessed.

let globalVar = "outside";

function test(){
  let localVar = "inside";
}
Scope Meaning
Global scope Accessible everywhere
Local / function scope Accessible only inside function
Block scope Accessible only inside block with let/const

15. Hoisting

Hoisting means declarations are moved to the top of their scope before execution.

console.log(x);
var x = 10;
Hoisting works differently for var, let, and const.

16. this Keyword

The this keyword refers to the current object.

let person = {
  name: "Shubham",
  greet: function(){
    console.log(this.name);
  }
};

In object methods, this points to the object itself.

17. DOM Manipulation

DOM means Document Object Model. It allows JavaScript to interact with HTML.

const heading = document.querySelector("h1");
const allCards = document.querySelectorAll(".card");

heading.textContent = "New Title";
heading.style.color = "#facc15";
heading.classList.add("active");
Method Use
querySelector() Select first matching element
querySelectorAll() Select all matching elements
textContent Change text
innerHTML Change HTML inside element
classList Add, remove, toggle classes

18. Events

Events happen when users interact with the page.

const btn = document.querySelector("#btn");

btn.addEventListener("click", () => {
  alert("Button clicked!");
});
document.addEventListener("keydown", (e) => {
  console.log(e.key);
});
Event Meaning
click Mouse click
keydown Keyboard key pressed
scroll Page scrolling
submit Form submitted
input User types into field

19. Classes and Objects

Classes are blueprints for creating objects.

class Student{
  constructor(name, age){
    this.name = name;
    this.age = age;
  }

  greet(){
    return `Hello ${this.name}`;
  }

const s1 = new Student("Shubham", 20);

constructor runs when object is created.

method is a function inside class.

21. localStorage

localStorage stores data in the browser.

localStorage.setItem("name", "Shubham");
let user = localStorage.getItem("name");
localStorage.removeItem("name");
localStorage.clear();
localStorage keeps data even after page reload.

22. Asynchronous JavaScript

Async JavaScript handles tasks that take time, like fetching data or waiting.

setTimeout(() => {
  console.log("After 2 seconds");
}, 2000);
console.log("1");
setTimeout(() => console.log("2"), 0);
console.log("3");
Concept Meaning
setTimeout Runs code after delay
setInterval Repeats code at interval
Promise Represents future result
async/await Simpler way to work with promises

23. Fetch API

Fetch API is used to get data from servers.

fetch("https://api.example.com/data")
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.log(error));
async function loadData(){
  try{
    let response = await fetch("https://api.example.com/data");
    let data = await response.json();
    console.log(data);
  }catch(error){
    console.log(error);
  }
}

Fetch is very important for API work and real-world applications.

24. Error Handling

Use try and catch to handle errors safely.

try{
  let result = x + 10;
  console.log(result);
}catch(error){
  console.log("Error:", error);
}
Part Use
try Code that may fail
catch Handle the error
finally Runs no matter what

25. Modules

Modules help split code into separate files.

export function greet(){
  return "Hello";
}
import { greet } from "./utils.js";
Modules make code cleaner and easier to manage.

26. JavaScript Best Practices

  • Use let and const instead of var.
  • Use meaningful variable names.
  • Keep functions short and focused.
  • Use strict equality === when possible.
  • Handle errors properly.
  • Keep code readable and organized.
  • Use semicolons consistently.
  • Separate HTML, CSS, and JS files when possible.

27. Important JavaScript Interview Points

Question Answer
What is JavaScript? JavaScript is a programming language used to make web pages interactive.
Difference between let, const, and var? let and const are block scoped, var is function scoped.
What is the DOM? The DOM is the structure of a web page that JavaScript can manipulate.
What is hoisting? Declarations are moved to the top of scope before execution.
What is the difference between == and ===? == checks value only, === checks value and type.
What is async/await? A cleaner way to work with asynchronous code.
Learn variables, functions, loops, arrays, objects, DOM, events, async code, and APIs to become strong in JavaScript.

28. Full JavaScript Example

This example shows a simple interactive page using JavaScript.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>JS Example</title>
</head>
<body>
  <h1 id="title">Hello</h1>
  <button id="btn">Click Me</button>

  <script>
    const title = document.getElementById("title");
    const btn = document.getElementById("btn");

    btn.addEventListener("click", () => {
      title.textContent = "Button Clicked!";
      title.style.color = "gold";
    });
  </script>
</body>
</html>

Quick recap: JavaScript makes websites interactive, dynamic, and powerful.
