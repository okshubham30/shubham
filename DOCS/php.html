1. Introduction to PHP

PHP stands for Hypertext Preprocessor. It is a server-side scripting language used to build dynamic websites, forms, login systems, dashboards, and web applications.

HTML gives structure

CSS gives design

JavaScript gives interactivity

PHP works on the server and handles logic, data, and database tasks

PHP is very popular for backend development and is widely used with MySQL.

2. PHP Setup

To run PHP code, you need a local server like XAMPP, WAMP, MAMP, or a hosting server like Hostinger.

XAMPP

Popular local server package with Apache and MySQL.

htdocs folder

Place PHP files here in XAMPP.

localhost

Used to open PHP pages in the browser locally.

C:\xampp\htdocs\project\index.php

3. PHP Syntax

PHP code is written inside special tags.

<?php
  echo "Hello PHP";
?>
Every PHP statement usually ends with a semicolon ;.

You can mix PHP with HTML in the same file.

<html>
<body>
  <h1><?php echo "Welcome"; ?></h1>
</body>
</html>

4. Variables

Variables are used to store values like names, numbers, and data.

<?php

Rule Meaning

Case-sensitive $name and $Name are different

7. Conditional Statements

Conditions help PHP make decisions.

<?php
$age = 18;

if ($age >= 18) {
    echo "Adult";
} else {
    echo "Minor";
}
?>

elseif

<?php
$marks = 85;

if ($marks >= 90) {
    echo "A+";
} elseif ($marks >= 75) {
    echo "A";
} else {
    echo "B";
}
?>

switch

<?php
$day = "Monday";

switch ($day) {
    case "Monday":
        echo "Start of week";
        break;
    case "Friday":
        echo "Weekend is near";
        break;
    default:
        echo "Normal day";
}
?>

8. Loops

Loops repeat code again and again.

for loop

<?php
for ($i = 1; $i <= 5; $i++) {
    echo $i . "<br>";
}
?>

while loop

<?php
$i = 1;
while ($i <= 5) {
    echo $i . "<br>";
    $i++;
}
?>

do while loop

<?php
$i = 1;
do {
    echo $i . "<br>";
    $i++;
} while ($i <= 5);
?>

foreach loop

<?php
$colors = ["Red", "Blue", "Green"];

foreach ($colors as $color) {
    echo $color . "<br>";
}
?>

9. Functions

Functions help you reuse code and keep your program clean.

<?php
function greet() {
    echo "Hello";
}

greet();
?>

Function with parameter

<?php
function greet($name) {
    echo "Hello " . $name;
}

greet("Shubham");
?>

Function with return value

<?php
function add($a, $b) {
    return $a + $b;
}

echo add(5, 3);
?>
Functions make code reusable, easier to read, and easier to debug.

10. Arrays

Arrays store more than one value in a single variable.

Indexed array

<?php
$fruits = ["Apple", "Mango", "Banana"];
echo $fruits[0];
?>

Associative array

<?php
$student = [
    "name" => "Shubham",
    "age" => 20
];

echo $student["name"];
?>

Multidimensional array

<?php
$students = [
    ["Shubham", 20],
    ["Rahul", 21]
];

echo $students[0][0];
?>
Function Use
count() Count array elements
array_push() Add item to end
array_pop() Remove last item
sort() Sort array

11. Forms and User Input

Forms are used to collect data from users.

<form method="POST">
  <input type="text" name="username" placeholder="Enter name">
  <button type="submit">Submit</button>
</form>
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $username = $_POST["username"];
    echo "Hello " . $username;
}
?>

GET sends data through the URL.

POST sends data hidden from the URL.

$_POST receives form data.

13. Sessions

Sessions store information on the server and are used for login systems.

<?php
session_start();
$_SESSION["user"] = "Shubham";
?>
<?php
session_start();
echo $_SESSION["user"];
?>
<?php
session_start();
session_destroy();
?>
Start the session using session_start() before reading or writing session variables.

14. Cookies

Cookies store small data in the browser.

<?php
setcookie("user", "Shubham", time() + 3600);
?>
<?php
echo $_COOKIE["user"];
?>
Cookies are useful but should not be used for sensitive data like passwords.

16. File Upload

File upload is used to upload images, documents, and other files.

<form method="POST" enctype="multipart/form-data">
  <input type="file" name="myfile">
  <button type="submit">Upload</button>
</form>
<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    move_uploaded_file(
        $_FILES["myfile"]["tmp_name"],
        "uploads/" . $_FILES["myfile"]["name"]
    );
    echo "File uploaded successfully";
}
?>
Always validate file type and size before uploading in real projects.

17. PHP and MySQL

PHP is often used with MySQL to store and manage data.

Database connection

<?php
$conn = mysqli_connect("localhost", "root", "", "testdb");

if (!$conn) {
    die("Connection failed: " . mysqli_connect_error());
}
?>

Insert data

<?php
$sql = "INSERT INTO students (name, age) VALUES ('Shubham', 20)";
mysqli_query($conn, $sql);
?>

Select data

<?php
$sql = "SELECT * FROM students";
$result = mysqli_query($conn, $sql);

while ($row = mysqli_fetch_assoc($result)) {
    echo $row["name"] . "<br>";
}
?>

Update data

<?php
$sql = "UPDATE students SET name='Rahul' WHERE id=1";
mysqli_query($conn, $sql);
?>

Delete data

<?php
$sql = "DELETE FROM students WHERE id=1";
mysqli_query($conn, $sql);
?>

19. Object-Oriented Programming

OOP makes PHP code more organized, reusable, and scalable.

Class and object

<?php
class Student {
    public $name;

    public function sayHello() {
        echo "Hello from Student";
    }
}

$obj = new Student();
$obj->name = "Shubham";
$obj->sayHello();
?>

Constructor

<?php
class Person {
    public $name;

    public function __construct($name) {
        $this->name = $name;
    }
}

$p = new Person("Shubham");
echo $p->name;
?>

Inheritance

<?php
class ParentClass {
    public function hello() {
        echo "Hello Parent";
    }
}

class ChildClass extends ParentClass {
}

$obj = new ChildClass();
$obj->hello();
?>

Access modifiers

Modifier Meaning
public Accessible everywhere
private Accessible only inside class
protected Accessible inside class and child class

20. PHP Security

Security is very important in real websites and applications.

Input validation

Always check user input before using it.

SQL injection protection

Use prepared statements instead of direct query strings.

Password hashing

Use password_hash() and password_verify().

Output escaping

Use htmlspecialchars() for safe output.

<?php
$passwordHash = password_hash("123456", PASSWORD_DEFAULT);

if (password_verify("123456", $passwordHash)) {
    echo "Password matched";
}
?>
Never store plain text passwords in a real project.
