1. Introduction to CSS

CSS means Cascading Style Sheets. It is used to style HTML elements and control the visual appearance of a web page.

HTML gives structure

CSS gives style

JavaScript gives behavior

CSS controls colors, fonts, spacing, layout, animations, responsiveness, and much more.

2. CSS Syntax

CSS follows a simple pattern: selector, property, and value.

selector{
  property: value;
}
p{
  color: red;
  font-size: 20px;
}
Part Meaning
Selector Targets HTML element
Property What you want to change
Value The new setting

3. CSS Selectors

Selectors are used to target HTML elements for styling.

*{
  margin: 0;
  padding: 0;
}

h1{
  color: red;
}

.box{
  background: black;
}

#hero{
  color: white;
}

h1,p{
  color: blue;
}
Selector Meaning Example
Universal All elements *
Element By tag name h1
Class By class .box
ID By unique id #hero
Group Multiple selectors h1,p

4. Colors

CSS supports many color formats.

color: red;
color: #38bdf8;
color: rgb(255, 0, 0);
color: rgba(255, 0, 0, 0.5);
color: hsl(200, 100%, 50%);

Named color

Example: red, blue, green

HEX

Example: #38bdf8

RGB / RGBA

Red, Green, Blue values with optional alpha

HSL

Hue, Saturation, Lightness

5. Backgrounds

Background properties control the background of an element.

background: red;
background-image: url(bg.jpg);
background-size: cover;
background-repeat: no-repeat;
background-position: center;

cover makes the image fit the whole box.

center centers the background image.

6. Text Styling

CSS gives full control over text appearance.

font-size: 20px;
font-weight: bold;
font-family: Arial, sans-serif;
text-align: center;
text-transform: uppercase;
letter-spacing: 2px;
line-height: 1.6;
Property Use
font-size Controls text size
font-weight Controls boldness
text-align Left, center, right
text-transform Uppercase or lowercase
letter-spacing Space between letters

7. Box Model

Every element is a box in CSS.

.box{
  width: 300px;
  padding: 20px;
  border: 2px solid white;
  margin: 30px;
}
Part Meaning
Content Actual text or image
Padding Space inside border
Border Line around content
Margin Space outside border
Box model is one of the most important CSS concepts.

8. Width and Height

Width and height control the size of elements.

width: 300px;
height: 200px;
max-width: 1200px;
min-height: 100vh;

max-width helps responsive design.

min-height is useful for full-screen sections.

9. Display Property

The display property defines how an element behaves in layout.

display: block;
display: inline;
display: inline-block;
display: none;
display: flex;
display: grid;
Value Meaning
block Takes full width
inline Stays in line
inline-block Inline but supports size
none Hidden
flex Flexible 1D layout
grid Grid 2D layout

10. Position Property

Position controls how an element is placed on the page.

position: static;
position: relative;
position: absolute;
position: fixed;
position: sticky;
.navbar{
  position: fixed;
  top: 0;
  left: 0;
  right: 0;
}
Type Meaning
static Default position
relative Moves from its normal place
absolute Moves relative to nearest positioned parent
fixed Stays fixed on screen
sticky Switches between relative and fixed

11. Flexbox

Flexbox is used for one-dimensional layouts.

.container{
  display: flex;
  justify-content: center;
  align-items: center;
  flex-direction: row;
  gap: 20px;
  flex-wrap: wrap;
}
Property Purpose
justify-content Aligns along main axis
align-items Aligns along cross axis
flex-direction Row or column
gap Space between items
flex-wrap Allows items to wrap
Use Flexbox for navbars, centering, rows of cards, and alignment.

12. CSS Grid

Grid is used for two-dimensional layouts.

.grid{
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
.grid{
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
}
Grid Feature Use
grid-template-columns Defines columns
grid-template-rows Defines rows
gap Space between grid items
grid-column Span across columns
grid-row Span across rows

13. CSS Units

Units are used to define sizes.

Unit Meaning
px Pixels
% Percentage of parent
em Relative to parent font-size
rem Relative to root font-size
vw Viewport width
vh Viewport height 
font-size: 2rem;
width: 100%;
max-width: 1200px;

14. Responsive Design

Responsive design makes websites look good on all screen sizes.

@media (max-width: 768px){
  .grid{
    grid-template-columns: 1fr;
  }
}
@media (min-width: 768px){
  .container{
    padding: 32px;
  }
}

Mobile first means designing for small screens first.

Then you expand styles for larger screens using media queries.

15. CSS Variables

Variables store reusable values.

:root{
  --primary: #38bdf8;
  --gap: 16px;
}

h1{
  color: var(--primary);
}

Benefits

Easy theme changes and less repeated code.

Where to use

Colors, spacing, sizes, and layout values.

16. Pseudo Classes

Pseudo classes style elements based on state.

button:hover{
  background: red;
}

input:focus{
  outline: none;
}

li:first-child{
  color: red;
}
Pseudo Class Use
:hover When mouse is over element
:focus When element is focused
:first-child First item inside parent
:nth-child() Select based on order

17. Pseudo Elements

Pseudo elements style parts of elements.

h1::before{
  content: "🔥";
}

h1::after{
  content: "🚀";
}
Pseudo Element Use
::before Add content before element
::after Add content after element
::first-line First line of text
::first-letter First letter of text

18. Transitions

Transitions make changes smooth instead of instant.

button{
  transition: .3s;
}

button:hover{
  transform: scale(1.1);
}
Transitions are often used for hover effects and UI interactions.

19. Animations

Animations are used for moving or changing elements over time.

@keyframes fade{
  from{
    opacity: 0;
  }
  to{
    opacity: 1;
  }
}

.box{
  animation: fade 1s ease;
}
Part Use
@keyframes Defines animation steps
animation-name Animation name
animation-duration Animation speed
animation-timing-function Animation style

20. Transform

Transform changes the shape or position of an element.

transform: translateX(50px);
transform: rotate(45deg);
transform: scale(1.2);
transform: skew(10deg);

translate moves element

rotate rotates element

scale resizes element

skew tilts element

21. Shadow and Overflow

Shadows and overflow improve visuals and control content spill.

box-shadow: 0 10px 30px rgba(0,0,0,.2);
text-shadow: 2px 2px 5px black;

overflow: hidden;
overflow: auto;
overflow: scroll;
Property Use
box-shadow Add shadow to boxes
text-shadow Add shadow to text
overflow Controls extra content

22. CSS Best Practices

  • Use classes more than IDs for styling.
  • Keep CSS organized and readable.
  • Use Flexbox and Grid for layouts.
  • Use variables for repeated values.
  • Design mobile first.
  • Avoid overusing !important.
  • Use meaningful class names.
  • Write reusable and clean CSS.

23. Important CSS Interview Points

Question Answer
What is CSS? CSS is used to style HTML elements.
Difference between Flexbox and Grid? Flexbox is 1D layout, Grid is 2D layout.
What is the Box Model? Content + Padding + Border + Margin.
What is Responsive Design? Making pages work on all screen sizes.
What are CSS Variables? Reusable custom values in CSS.
Master selectors, box model, flexbox, grid, responsive design, transitions, and animations to become strong in CSS.

24. Full CSS Example

This example shows a simple styled webpage using common CSS properties.

body{
  margin: 0;
  font-family: Arial, sans-serif;
  background: #0f172a;
  color: white;
}

.header{
  background: #111827;
  padding: 20px;
  text-align: center;
}

.container{
  display: flex;
  justify-content: center;
  gap: 20px;
  padding: 30px;
}

.card{
  background: #1e293b;
  padding: 20px;
  border-radius: 15px;
  box-shadow: 0 10px 30px rgba(0,0,0,.2);
}

@media (max-width: 768px){
  .container{
    flex-direction: column;
  }
}

Quick recap: CSS controls the look, layout, spacing, and responsiveness of a website.
